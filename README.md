# 🎂 Best Bakes

Best Bakes is a full-stack bakery e-commerce website where users can browse cakes, view product details, add items to their cart, and enjoy a seamless online cake shopping experience.

## 🌐 Live Website

**Frontend:** https://best-bakes-frontend.vercel.app

**Backend API:** https://best-bakes-backend.onrender.com

---

## ✨ Features

- 🍰 Browse a variety of delicious cakes
- 📋 View detailed product information
- 🛒 Add and manage items in the shopping cart
- 💾 Dynamic product loading from MongoDB
- 👤 User login page
- 📱 Fully responsive design
- ⚡ Fast and interactive user experience

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap
- AOS (Animate On Scroll)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

### Deployment
- Frontend: Vercel
- Backend: Render

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/ancysneha/best-bakes-frontend.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
npm start
```

### Run the Frontend

Open `index.html` in your browser or deploy it using Vercel.

---

## 🌍 API Endpoint

```
GET /api/products
```

Fetches all available cake products from the MongoDB database.
